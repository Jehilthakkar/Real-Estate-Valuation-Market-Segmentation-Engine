# Real-Estate-Valuation-Market-Segmentation-Engine
The project Real Estate Valuation Market Segmentation Engine primarily deals with House price prediction. When we hear about house price prediction the first comes to our mind is linear regression, but it works only when the amount of data is less. I have used 'Ames housing dataset' to design this project, this dataset contains exactly 82 columns which is very high when training any model. Feeding this huge amount of data to a model like linear regression is very risky and also the prediction will be faulty, so I have used multiple algorithm including esemble techniques and also clustering algorithms to check which model suits well to this particular dataset.
